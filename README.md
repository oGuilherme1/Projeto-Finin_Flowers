<h4>Para instalar a aplicaçao em sua maquina siga os passos abaixo</h4>


```
git clone https://github.com/gsocria/Projeto-Finin_Flowers.git
```

```
composer install
```

```
php artisan serve --port=8000
```

Acessar a pagina  http://localhost:8000
